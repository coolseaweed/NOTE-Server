## 1. Github commands <a name="1.-Github-commands"></a>

  ### 코드 업로드
  ```bash
  $ git status 
  $ git add <file/directory> 
  $ git commit -m "message"
  $ git push 
  
  # commit cancel
  $ git reset HEAD ^  
  ```
  
  ### git repo 와 local dir 연동
  ```  
  
  
  $ git remote rm origin
  $ git init
  
  
  ```
