## 1. 필수 package <a name="1.-NVIDIA-driver-설치"></a>

  ### GPU info 확인
  ```bash
  $ sudo lshw -C display # 그래픽카드 정보 확인
  $ ubuntu-drivers devices # 추천 드라이버 확인
  ```
  ### Ubuntu 16.04 LTS 
  나중에 깔게 되면 찾아서 넣음
